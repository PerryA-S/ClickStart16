Portfolio


Here are a few projects I’ve worked on:



  • Personal Finance Tracker - Developed a personal finance tracker application using Python and SQLite to help users manage their budget, track expenses, and set savings goals. The app provides monthly and yearly reports, visualizes spending patterns, and sends reminders to stay on track with budgets. This project helped me improve my skills in data manipulation, working with databases, and developing user-friendly command-line interfaces.


  • Weather Data Analysis Tool - Built a Python tool that scrapes weather data from various public APIs and visualizes trends and forecasts using Matplotlib and Pandas. The application allows users to view weather patterns over different periods and locations, providing insights into seasonal changes and climate data. This project deepened my understanding of web scraping, API integration, and data visualization.


  • Task Automation Script for Daily Workflows - Created an automation script using Python to streamline daily repetitive tasks such as file organization, email sorting, and data backups. The script improves efficiency by automating time-consuming workflows, allowing users to focus on more important tasks. This project enhanced my knowledge of Python libraries like os, shutil, and smtplib, and helped me understand the practical applications of automation in a real-world context.
